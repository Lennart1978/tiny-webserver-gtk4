<p><h1>Tiny-Webserver-GTK4</h1></p>
<p>A tiny webserver with GTK4 GUI written in C</p>
<br><i>
mkdir builddir &&
meson setup builddir &&
meson compile &&
sudo ./webserver-gtk
<br></i><br>
<p>Put startpage "index.html" in the same folder of webserver-gtk executable.</p>
